# Task_Buddy
A lightweight task manager application featuring priority-based task organization, status tracking (pending/completed), and dynamic filtering. Uses local Storage to maintain data across sessions. A simple and interactive task manager web app.

## Features ##
- Add, delete and complete tasks
- Priority-based sorting (High, Medium, Low)
- Filter tasks (All, Completed, Pending)
- Data saved using localStorage
